Overview & Vision

C.O.R.I. is a modular, ROS 2 based home assistant robot designed to automate household tasks like laundry sorting. This personal R&D project focuses on building a foundation for real-world autonomy and home integration, creating a flexible platform for learning and prototyping practical home robotics.

Key Features

  • ROS 2 Native: Built for modularity and scalability
  • Gazebo Simulation: Mobile base with camera simulation
  • Computer Vision: OpenCV HSV filtering for laundry sorting
  • Expandable: Architecture for navigation, manipulation, and task planning
